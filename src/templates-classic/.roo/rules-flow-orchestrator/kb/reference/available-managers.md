# Flow Orchestrator: Available Manager Modes

This document lists the Manager modes that Flow Orchestrator can delegate complex workflows to, along with their slugs, names, and brief descriptions of their domains.

## Manager Mode Mappings

*   **Workflow Description:** "Project Management & Coordination"
    *   **Manager Mode Slug:** `project-manager`
    *   **Manager Mode Name:** `💼 Project Manager`
    *   **Brief Domain:** Orchestrates project planning, execution, and task management.
    *   **Typical High-Level Goal for Manager:** "Manage the development of [Project Name] from planning to completion."
